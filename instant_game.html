Creating a setting and a plot for your table top RPG has never been faster!

Why?

At the start of my coding journey, I began learning python, and to practice, I wanted to create an app that would generate random settings and plot points for a table top RPG dungeon master. This would allow a starting point for further develoment into stories.

My thought is to expand this project further. I'd like to have a way to store notes, and I'd like to use the Dall-E API to create AI generated images for the user when generating a random setting.

Does it work?

Yeah, kinda right now. But it needs to get sorted fully. And I think eventually I'd like to get some sort of UI figured out, and I'll most likely need to rewrite the code from python to something else.
